# Bento RSVP

Projeto React + Vite pronto para rodar.

## Como usar

```bash
npm install
npm run dev
```

## Foto

Troque o arquivo `public/foto-crianca.jpg` pela foto do aniversariante mantendo o mesmo nome.

## Arquivos principais para editar

- `src/App.jsx` -> textos e estrutura da pagina
- `src/index.css` -> cores, layout e animacoes
- `public/foto-crianca.jpg` -> foto redonda
